# BigData-Mining-Practices
Big Data Collection and Mining (Capstone Design) course labs using Python. 
Focuses on public data analysis, visualization (Matplotlib), and data manipulation (NumPy, Pandas, Web Crawling).

파이썬을 이용한 공공데이터 분석, Matplotlib 기반 데이터 시각화, NumPy/Pandas를 활용한 데이터 처리 및 웹 크롤링 프로젝트를 포함합니다.
